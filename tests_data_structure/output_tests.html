Model output test

1 Introduction

In this document, we will implement two methods to store the model output. The first method will be a list of tibble, the second method is a single tibble. First there will be a brief description of each method. Then a brief explication of what each method do and how they works with the results they give. Then we will do some tests in order to decide which method is the best for the model output storage.
